# DataDigger
Data Digger – SQL Project

A simple MySQL project demonstrating CRUD operations, joins, aggregate functions, and relational database design.

Files
schema.sql – Creates database and tables
seed.sql – Inserts sample data
README.md – Project documentation

Tables
Purchasers
Products
Orders
OrderDetails

How to Run
Install MySQL.
Run in MySQL client:

SOURCE schema.sql;
SOURCE seed.sql;
SOURCE queries.sql;


(Optional) Edit app.py with DB credentials and run:
python app.py
Features
Auto-increment primary keys
Foreign key relationships
CRUD operations
Joins

Aggregates (SUM, AVG, MAX, MIN)

Realistic sample data
